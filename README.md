# Project Name: Quantum-Proof Systems Scanner

Description:
A cybersecurity platform for analyzing cryptographic posture of public-facing applications and evaluating readiness for Post-Quantum Cryptography.

Features:

* TLS Security Scanning
* Certificate Analysis
* CBOM Generation
* PQC Readiness Evaluation
* Cyber Risk Analytics
* Reporting System
* Two-Factor Authentication

Tech Stack:

* Python
* Django
* Bootstrap
* Chart.js
* SQLite

Learning Goals:

* Applied Cryptography
* Cybersecurity Engineering
* Secure System Design
* Security Analytics

# 5. Portfolio Description

Developed a cybersecurity platform that analyzes cryptographic configurations of public-facing applications and evaluates readiness for Post-Quantum Cryptography. Implemented TLS scanning, certificate analysis, CBOM generation, risk analytics, reporting, and two-factor authentication using Python and Django. The project provided hands-on experience in applied cryptography, secure software development, and cybersecurity system design.
